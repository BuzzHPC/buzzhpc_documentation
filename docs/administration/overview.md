Comprehensive administration capabilities for managing users, workspaces, permissions, and platform configuration.

---

## Administration Features

### User Management

- Create and manage user accounts
- Assign roles and permissions
- Enable/disable user access
- Reset passwords
- View user activity

### Workspace Management

- Create and organize workspaces
- Assign users to workspaces
- Set workspace quotas
- Configure workspace policies

### Access Control

- Role-based access control (RBAC)
- Permission management
- API key generation
- SSH key management

### Audit Trail

- View system activity logs
- Search and filter events
- Export audit reports
- Compliance monitoring

### Settings

- Platform configuration
- Integration settings
- Notification preferences
- Billing settings

Navigate to **Administration** in the left menu to access administrative functions.
